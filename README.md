# Manufacturing-Dashboard-Predictive-Alerts
Real-time manufacturing monitoring and predictive maintenance dashboard using Python and machine learning.


###  Tech Stack
- Python, SQL Server, Power BI / Grafana  
- Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn  

###  Functional Features
- Collects live machine data (temperature, vibration, speed, runtime)
- Predicts failures and sends alerts
- Interactive dashboard visualization
- Historical data storage and trend analysis

###  Model Performance
- **Accuracy:** 100%
- **Confusion Matrix:** No false positives/negatives
- Predictive alerts for abnormal sensor readings

### Business Impact
- Reduces downtime through predictive maintenance
- Improves manufacturing efficiency
- Aligns with Industry 4.0 smart factory goals

###  Files
- `manufacturing_dashboard.ipynb` – Jupyter/Colab notebook  
- `predictive_model.pkl` – Trained ML model  
- `outputs/` – Visualization and evaluation results  
- `report/` – PDF project report
